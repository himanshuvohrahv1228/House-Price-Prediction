# 🏠 House Price Predictor

A simple Machine Learning project built using Python and Streamlit to predict house prices based on user input.



## 🚀 Features

- Predicts house price based on:
  - Area (sq ft)
  - Number of bedrooms
- Interactive user interface using Streamlit
- Real-time prediction



## 🛠 Tech Stack

- Python
- Pandas
- Scikit-learn
- Streamlit



## 📂 Project Structure
House-Price-Predictor/
│
├── app.py
├── requirements.txt
└── README.md



## ▶️ How to Run

1. Install dependencies:

pip install -r requirements.txt


2. Run the app:

streamlit run app.py
